# Web mobile automation with Nightwatch.js and Appium

Source code for http://www.genbetadev.com/herramientas/automatizacion-de-pruebas-web-moviles-appium-nightwatch-js

Example of how to use Nightwatch.js automation framework combined with Appium for web automated testing.

![alt tag](https://i.blogs.es/f77cc4/proyectoenatom/1366_2000.png)

We need to have installed node.js, nightwatch.js and appium.

To start the test, from the command line, we run 'nightwatch -c config/nightwatch -e android'
